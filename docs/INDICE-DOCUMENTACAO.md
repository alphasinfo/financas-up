## ⚡ Início Rápido

### Instalação

```bash
# Linux (Manjaro/Arch)
./scripts/manjaro/install.sh

# Linux (Debian/Ubuntu)
./scripts/debian/install.sh

# Windows
.\scripts\windows\install.ps1
```

### Uso Básico

```bash
# Iniciar servidor
npm run dev

# Acessar
http://localhost:3000
```
